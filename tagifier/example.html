Simplest example

  • The tag area grows as needed if you add more tags than it can show
  • If you try to enter the same item twice, it will fail and the previous one will blink to show where it is
  • If you click on an item, you will be able to edit it
  • You can remove a tag by pressing the X in it, or remove the last one by pressing backspace with an empty input
  • Data will be returned as a json encoded array of all tags
